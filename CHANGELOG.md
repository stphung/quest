# Changelog

## [0.1.0] - 2026-02-01

### Features

- Terminal-based idle RPG with automatic progression
- 6 RPG attributes (STR, DEX, CON, INT, WIS, CHA)
- Dynamic combat system with scaling enemies
- Prestige system with XP multipliers
- 5 unique zones (Meadow to Volcanic Wastes)
- Offline progress tracking (50% rate, max 7 days)
- Auto-save every 30 seconds
- Binary save files with SHA256 checksums
- 3D ASCII first-person dungeon renderer
- Visual combat effects
- One-line installers for all platforms

### Development

- Comprehensive CI/CD pipeline with quality checks
- Format checking, linting, testing, building, security audits
- Multi-platform release builds (Linux, macOS x86/ARM, Windows)
- Unified local and CI check script
- Development Makefile for common tasks
